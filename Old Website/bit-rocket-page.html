Bit Rocket

iOS Game

Development Engine: Unity

Programming Language: C#

Bit Rocket is a game I released in early 2019. It is the first game I developed and released. It is a small, simple game with infinite gameplay. The game was developed in Unity, and was made with assets that I designed and made. It is a free game which is supported with unobtrusive advertisements.

Features

- Unity Ads integration

- Retro pixel art style

- Satisfying animations

- High score system
